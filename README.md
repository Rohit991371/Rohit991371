## 🧠 Featured Work

### 📄 [COVID-19 CT Scan Classification using Xception with Attention Mechanism](https://github.com/Rohit991371/Lungs-CT-scans-Classification)
*First author — IEEE, 2nd International Conference on Circuits, Power, and Intelligent Systems (CCPIS), 2025*
Achieved **97.86% accuracy** and **96.82% precision** on a medical imaging dataset using transfer learning, attention mechanisms, and data augmentation.
`Python` `TensorFlow` `Keras` `Transfer Learning` `Attention Mechanisms`

### 🤖 [InsightFlow AI — Multi-Agent Analytics Platform](https://github.com/Rohit991371/InsightFlow-AI)
An autonomous analytics platform built with LangGraph and LangChain that converts raw datasets into visualizations, business insights, and executive reports using 8 specialized AI agents for data cleaning, statistical analysis, chart explanation, and conversational analytics.
`Python` `LangGraph` `LangChain` `Streamlit` `LLMs`

### 🔍 [Personalized AI Assistant — RAG-Based Intelligent Agent](https://github.com/Rohit991371/Personalized-Agent)
A production-style Retrieval-Augmented Generation agent integrating multi-source personal data (resume, GitHub, portfolio) using embeddings, a FAISS vector store, and an LLM backend for context-aware responses. **[Live demo →](https://rohit991371-personalized-agentapp-5sqmdf.streamlit.app/)**
`Python` `LangChain` `FAISS` `Vector Search` `LLMs`

### 🔬 Two-Higgs-Doublet Model (2HDM) Exploration(https://github.com/Rohit991371/Two-Higgs-Doublet-Model)
*Research Internship — IIT Gandhinagar*
Engineered and processed 25 high-dimensional datasets (500K rows each) for particle physics parameter-constraint simulations; designed a regularized neural network achieving a **15% accuracy improvement** over baseline.
